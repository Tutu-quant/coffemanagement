🚀 HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

1. START THE APPLICATION
   - Press F5 in Visual Studio
   - OR: dotnet run
   - App launches on http://localhost:5269

2. LOGIN WITH ADMIN ACCOUNT
   - Username: admin
   - Password: admin123

3. NAVIGATE TO CATEGORIES
   - Click "Danh mục" in sidebar
   - OR: http://localhost:5269/Admin/Categories

4. MANAGE CATEGORIES
   - View: List with search & pagination
   - Create: Click "+ Thêm danh mục"
   - Edit: Click Edit button on any item
   - View Details: Click View button
   - Delete: Click Delete button (requires confirmation)

5. FEATURES TO EXPLORE
   - Search bar (top of list)
   - Pagination (bottom of list)
   - Status badges (Active/Inactive)
   - Breadcrumb navigation (top)
   - User profile (top right)
   - Sidebar menu
